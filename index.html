TECH O3

HASKELL

SYNOPSIS

TECH O3 introduces N2O protocols for Haskell programming language. The aim is to be compatible at transport level with Erlang version of N2O.

Stack

  • o3/n2o — N2O 0.11 HS
  • o3/nitro — NITRO 0.11
  • o3/sample — SAMPLE 0.11

N2O for Haskell
